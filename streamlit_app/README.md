# StyleVerse AI Streamlit App

Interactive AI personal stylist built with Streamlit.

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Optional Groq AI

Copy `.env.example` to `.env` and add:

```bash
GROQ_API_KEY=your_key_here
```

The app works without an API key using built-in styling rules.
